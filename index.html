Information Security & Video Encryption

LockVid Video Encryption System

Project Overview

An innovative capstone research and practical implementation resolving high resource overhead and computational latency within traditional video encryption. It introduces Block Scrambling mechanisms to safeguard colorful video streams over unstable networks without disrupting real-time streaming experiences or inducing visual decay.

Technical Implementations

  • Cryptographic Pipeline: Designed algorithms to seamlessly extract raw video frames, isolate primary RGB color channels, and apply automated Zigzag matrix scanning alongside randomized block rotations and multi-dimensional index shuffling.
  • Transform Domain Protection: Integrated dual mathematical frameworks using Discrete Cosine Transform (DCT) and Singular Value Decomposition (SVD) to project spatial data into frequency coefficients, vastly increasing system defenses against unauthorized brute-force decryption attacks.
  • Performance Analytics: Validated extreme security robustness by computing structural metrics, confirming exceptional Peak Signal-to-Noise Ratio (PSNR) values and strictly near-zero horizontal/vertical pixel correlation coefficients for uncompromised safety.

Technical Stack

MATLAB Image Processing Toolbox Transform Domain Cryptography GUI Dashboard Development

Medical Appointment Scheduling System

Project Overview

A back-end relational database solution designed to optimize hospital and clinic administration. The system securely automates scheduling, preserves patient-doctor constraints, and prevents overlapping reservations or scheduling conflicts through automated structural constraints.

Technical Implementations

  • Database Schema & Constraints: Engineered relational schemas with strict Primary and Foreign Key relational data mapping (Patients, Doctors, Appointments, and Users Tables).
  • Business Logic Automation: Implemented safe transaction handling using numeric Sequences for auto-increment keys and PL/SQL Triggers (e.g., PREVENT_LATE_BOOKING) to programmatically block same-doctor booking overlaps and prevent last-minute reservations (less than 15 minutes in advance).
  • Enterprise Deployment: Integrated and deployed data operational flows through an Oracle APEX (Application Builder) interface dashboard for administration and role-based access management.

Technical Stack

SQL / PL-SQL Oracle APEX Cloud Database Triggers & Sequences Data Integrity Constraints
