LockVid Video Encryption System

Graduation Project | MATLAB-Based Video Encryption

Project Overview

LockVid is a video encryption system developed as a graduation project. The project focuses on protecting video content using encryption techniques while maintaining quality and processing efficiency.

Technologies Used
